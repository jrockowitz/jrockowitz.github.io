A web-based content management system built for Memorial Sloan-Kettering Cancer Center.

The Inettool gives administrators and editors full control over a web-site's content and architecture.

Key Features

Administration Tools

A tool to manage the application, websites, and users settings.

  • Implements a workgroup/user based security model
  • Support for multiple website creation and management
  • Domain name and redirect management
  • Reporting and error logging
  • Customizable user interface based on different roles

Templating System

A flexible JavaScript based templating system.

  • Manage site-wide templates
  • Build custom banner templates

Site/Content Management

A tool to manage a website's information architecture and content.

  • Information architecture management including site maps and flow charts
  • Web-based custom rich text html editor
  • Content can be shared across multiple sections of a website
  • Syndication of multiple versions of the entire site including an html, printer-friendly, and text-only version
  • Dynamic display and removal of content based on user permissions and IP address.
  • Publishes static documents to server
  • Removed content is archived in removal bin
  • Built-in link management
  • Rich database search capabilities

Form Builder and Data Manager

A tool to build online forms that can send email and insert records into a database.

Community Management

A group of tools to organize and distribute news and information.

  • Send e-newsletters to users and employees
  • Create and manage mailing lists
  • Maintain a calendar of events
  • RSS/Podcast manager

Asset Management

A tool to organize images and documents.

  • Create slideshows
  • Build a file to download pages
  • Add meta data to file assets
